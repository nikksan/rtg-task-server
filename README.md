# Project Title

RTG Task: Server App


### Installing

A step by step series of examples that tell you how to get a development env running

Import the dummy database

```
mysql -u username -p database_name < acme_app_dev.sql
```

Install packages

```
npm install
```

Run the dev server

```
npm start
```

## Deployment
```
npm run production
```


## Running the tests
```
npm run test
```
